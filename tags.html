Important HTML tags

This page is design for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a new paragraph of text
table table include a table
tr table row incude a raw of data
th table header make a table header
td table data include a table data cell
div division creates a block of code
header header creates a header section
ol ordered list creates an ordered list
ul unordered list creates an unordered list

Inline elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make bold text <strong>technical sophistication</strong> technical sophistication
a anchor make hyperlink <strong>technical sophistication</strong> technical sophistication
code code make code text <a href="http://learnenough.com/">technical sophistication</a> technical sophistication
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
span span create a span section <span>technical sophistication</span> technical sophistication
li list element add one element to the list <li>technical sophistication</li>
